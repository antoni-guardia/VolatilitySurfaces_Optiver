# Optiver Seminar Volatility Surface  

## Purpose  
This project aims to demonstrate the construction and visualization of a volatility surface for options pricing. It provides insights into how volatility varies with different strikes and maturities and is essential for understanding options pricing dynamics.

## Project Structure  
- **/data**: Contains datasets used for modeling the volatility surface.  
- **/src**: Source code for constructing and visualizing the volatility surface.  
- **/notebooks**: Jupyter notebooks for exploratory data analysis and visualization.  
- **README.md**: This documentation file.  

## Dependencies  
- Python 3.8 or later  
- NumPy  
- Pandas  
- Matplotlib  
- SciPy  
- Jupyter Notebook  

You can install the necessary dependencies using pip:  
```
pip install numpy pandas matplotlib scipy jupyter
```

## Usage Instructions  
1. Clone the repository:  
   ```bash  
   git clone https://github.com/Lldelduca/optiver-seminar-volatility-surface.git  
   cd optiver-seminar-volatility-surface  
   ```  
2. Install the dependencies as mentioned above.  
3. Run the example scripts in the `/src` directory or open the notebooks in the `/notebooks` directory to explore the volatility surface visualization.
